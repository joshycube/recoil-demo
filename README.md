# RECOIL Demo

This is a demonstration of consuming a static API and managing the application state by Recoil.

## How to run this project to try it out?

### Prerequisites

- node
- git

### Steps

- `npm install`
- `npm run client`
- `npm run server`

## How to run tests on this project?

### Prerequisites

- cypress

### Steps

- `npm run client`
- `npm run server`
- `npm run cypress:open`
